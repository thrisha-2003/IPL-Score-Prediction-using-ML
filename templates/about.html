About This Application

Welcome to the IPL Score and Win Prediction Application. This application is designed to predict the score and match outcomes of IPL cricket matches based on various parameters such as teams, venue, current runs, current overs, and current wickets.

The application is built using FastAPI, a modern web framework for building APIs with Python 3.7+ based on standard Python type hints. It includes the following features:

  • User registration and authentication using OAuth2 and JWT tokens.
  • Prediction using Linear Regression
  • Responsive design with Bootstrap for a smooth user experience.

Technologies Used

  • FastAPI
  • SQLAlchemy
  • Jinja2 Templates
  • Bootstrap 4
  • Machine Learning models with scikit-learn

How to Use

To use the application, follow these steps:

  1. Register a new account on the registration page.
  2. Login with your credentials on the login page.
  3. Navigate to the prediction form and fill in the match details.
  4. Submit the form to get predictions for the match.
